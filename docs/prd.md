# 需求文档

## 1. 应用概述

### 1.1 应用名称
AI 设计发布会 PPT 生成网站

### 1.2 应用描述
一款基于 AI 图片生成技术的 PPT 制作工具,用户可通过文字描述生成图片和视频,并使用内置编辑器创建专业的 AI 设计发布会 PPT,支持多种模板选择、自定义尺寸和在线编辑导出功能。

### 1.3 设计风格
极简高级风格,对标 lovart.ai:
  - 极简留白:充足的页面间距
  - 低饱和高级配色:#1a1a1a 深灰 + #e5e5e5 浅灰 + #6b7280 中性灰
  - 轻奢商务:克制用色,专业感优先
  - 现代简约:去除装饰性元素,功能性设计
  - 干净无冗余:每个元素都有明确目的

## 2. 用户与使用场景

### 2.1 目标用户
  - 需要快速制作 AI 主题发布会 PPT 的产品经理
  - 科技公司市场营销人员
  - AI 行业从业者和演讲者

### 2.2 核心使用场景
  - 用户需要为 AI 产品发布会准备演示文稿
  - 用户希望通过 AI 生成图片和视频快速制作专业 PPT
  - 用户需要在线编辑和管理多个 PPT 作品

## 4. 业务规则与逻辑

### 4.1 AI 图片生成规则
  - 统一模型调度中枢 ModelService 类管理所有模型切换
  - GPT-Image-2 接口规范:
    + API 端点:https://grsai.dakka.com.cn/v1/draw/completions
    + 请求头:Authorization: Bearer {API_KEY}
    + 标准入参:model, prompt, size, response_format, n, style
    + 标准出参解析:data[0].url 获取图片 URL
    + 统一响应格式:{success: bool, url?: string, error?: string}
  - 错误处理机制:
    + 网络错误、API 密钥无效、请求超限、生成失败
    + 请求日志打印
    + 重试机制:最多重试 3 次
    + 接口超时:60 秒
  - Fallback 机制:主模型失败自动切换到备用模型
  - 支持模型:GPT-Image-2、通义万相、DALL·E 等

### 4.2 AI 视频生成规则
  - 统一模型调度中枢管理视频生成模型
  - 视频生成接口规范:
    + 标准入参:model, prompt, resolution, duration, style, fps, motionStrength, quality
    + 标准出参解析:获取视频 URL
    + 统一响应格式:{success: bool, url?: string, error?: string}
  - 错误处理机制:
    + 网络错误、API 密钥无效、请求超限、生成失败
    + 请求日志打印
    + 重试机制:最多重试 3 次
    + 接口超时:120 秒（视频生成时间较长）
  - Fallback 机制:主模型失败自动切换到备用模型

### 4.3 PPT 编辑规则
  - 用户可在编辑器中自由添加、编辑、删除幻灯片
  - 所有编辑操作实时保存到本地或服务器
  - 支持撤销和重做操作
  - 图层编辑、文字修改、元素拖拽正常使用
  - 幻灯片尺寸调整规则:
    + 预设比例切换时自动计算尺寸
    + 自定义尺寸输入后实时更新画布
    + 画布拖拽调整时显示当前尺寸数值
    + 尺寸变更后所有元素按比例自动调整位置
  - 图片/视频元素交互规则（基于 Fabric.js）:
    + 选中元素显示控制框和控制点
    + 拖拽移动时显示对齐辅助线
    + 四角拖拽调整大小,保持中心点不变
    + 按住 Shift 键拖拽时等比例缩放
    + 旋转控制点拖拽实现旋转
    + 对齐辅助线在元素与其他元素或画布边缘对齐时自动显示

### 4.4 作品保存规则
  - 用户创建的 PPT 自动保存到我的作品
  - 每个作品包含完整的幻灯片数据和元素信息
  - 支持导出为标准 PPTX 格式文件
  - 视频元素在导出时嵌入 PPTX 文件

### 4.5 用户登录状态
  - 登录状态本地存储
  - 用户未登录时仅可访问首页,其他页面需登录后访问

### 4.6 AI 发布会 PPT 生成流程
  - 输入:主题、行业、发布会场景、品牌调性、页数、风格、色调、文案要求
  - 输出:完整发布会 PPT 大纲、全套版式、统一视觉规范
  - 多风格切换:极简商务、国企稳重、高端发布会、科技未来、人文暖调、轻奢高级

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| API Key 未配置 | 提示用户前往设置页面配置 API Key |
| AI 图片生成失败 | 显示错误提示信息,允许用户重新生成,自动重试最多 3 次 |
| AI 视频生成失败 | 显示错误提示信息,允许用户重新生成,自动重试最多 3 次 |
| 接口请求超时 | 图片生成 60 秒超时,视频生成 120 秒超时,显示超时提示,允许重试 |
| 网络错误 | 显示网络错误提示,检查网络连接 |
| 上传文件格式不支持 | 提示仅支持 PNG、JPG、PDF 格式 |
| PPT 导出失败 | 显示错误提示,允许用户重试 |
| 用户未登录访问受限页面 | 跳转到登录页面 |
| 删除作品操作 | 弹出二次确认提示 |
| 模型切换失败 | 自动切换到备用模型,显示切换提示 |
| 模型测试连接失败 | 显示连接失败原因,提示检查配置 |
| 幻灯片尺寸超出合理范围 | 提示尺寸范围限制（宽度 100-10000px,高度 100-10000px） |
| 视频时长超出限制 | 提示仅支持 5/10/15 秒时长 |
| Fabric.js 加载失败 | 显示错误提示,降级为基础编辑模式 |
